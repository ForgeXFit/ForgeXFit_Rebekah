Primary Mission

Goal: controlled fat loss + glute growth + tighter waist/core while supporting breastfeeding and postpartum recovery.

  • Look target: slim thick, toned tummy, round glutes, posture, and feminine upper-body shape.
  • Rate of loss: ~0.5–1 lb/week is ideal.
  • Training goal: make 2 days count, not crush recovery.
  • Nutrition goal: enough food to train, recover, and support milk supply.

Postpartum Guardrails

Important: She is cleared to lose weight, but breastfeeding still changes the plan. No crash dieting, no aggressive cardio, no heavy bracing if core pressure feels wrong.

  • If milk supply drops → add 150–300 calories, mostly carbs.
  • If extreme fatigue or dizziness occurs → increase food and reduce training intensity.
  • If belly bulging/doming happens during core work → regress the core exercise.

Progression Rules

  • Use double progression: add reps first, then weight.
  • When she hits the top of the rep range with clean form, increase weight next time.
  • Most sets should stop with 1–2 reps in reserve.
  • Glute isolation work can be pushed harder if form stays clean.

Form Priorities

  • Hip thrust: ribs down, chin tucked, full glute squeeze.
  • RDL: hips back, hamstring stretch, neutral spine.
  • Abductors: no bouncing, pause slightly open.
  • Core: exhale, ribs down, no belly bulge.

Adjustment Rules

  • If weight drops faster than 1 lb/week and milk/energy suffer → add 150–300 calories.
  • If weight does not move for 2 weeks and milk is stable → reduce 100–150 calories or add walking.
  • If glutes are not progressing → keep calories higher and push hip thrust/RDL progression.
  • If recovery is poor → remove optional finisher first.
